iminuit¶

iminuit is a python (espcially ) friendly wrapper of lcg-minuit . The most popular minimizer used in High Energy Physics which has been around for more than 40 years(for a good reason).

iminuit is designed from ground up to be used in an interactive fitting environment taking full advantage of IPython notebook. To make transition from the popular PyMinuit easy, RTMinuit is (mostly) compatible with PyMinuit (just change the import statement).

Download¶

git clone git://github.com/piti118/iminuit.git

Install¶

python setup.py install
